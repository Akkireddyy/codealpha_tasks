# Language Translation Tool

## Overview
The Language Translation Tool is a responsive web application that allows users to translate text between multiple languages. The project is developed using HTML, CSS, and JavaScript, and integrates a translation API for real-time multilingual translation.

The application also includes additional features such as text-to-speech, copy translation, language swapping, responsive design, and modern UI enhancements.



## Features
* Translate text between multiple languages
* Real-time translation using Translation API
* Copy translated text to clipboard
* Text-to-Speech (TTS) support
* Swap source and target languages
* Clear text functionality
* Responsive design for desktop and mobile devices
* Modern animated user interface



## Technologies Used
* HTML5
* CSS3
* JavaScript
* Google Translation API (Unofficial Endpoint)
* Web Speech API

## How to Run the Project
1. Download or clone the repository
2. Open the project folder in Visual Studio Code
3. Install the Live Server extension
4. Right-click `index.html`
5. Select `Open with Live Server`
The project will open in your browser.
